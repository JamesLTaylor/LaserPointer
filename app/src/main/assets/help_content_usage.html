Start

To start sending mouse moves for changes in phone orientation press start. The button then changes to a Stop button which has the obvious effect.

Moving left and right

Roll the hand left and right. There are pictures of the hand position online.

The reason that a roll is required is that the accelerometer is used to determine the angle rather than the gyroscope.

Moving up and down

Moving the cursor up and down is exactly as you would expect, simply tilt the phone to point to the top or bottom of the screen.

Resetting the IP address

Type in the new IP address and press the reset button.

CTRL and left button down

Microsoft power point changes to the mouse cursor to a red dot when ctrl and the left button are held down. This switch achieves this for you. One must be a bit careful to turn it off when it is no longer required because your computer can be hard to work with if it acts as though the CTRL key is permanently held down.

Keep screen on

This forces the phone screen to stay on. Messages will stop being sent when the screen turns off.

Previous

Send the page up key to the computer. In most applications this will move you to the previous slide.

Next

Send the page down key to the computer. In most applications this will move you to the next slide.

Click

Send a left mouse click at the current cursor location. Can be useful for bringing the focus to the application with the presentation.

Calibrating

If you find that the default angles that you need to hold the phone at are unnatural or don't suit the angle of the actual screen that you are using then you can cange 1 or all of the angles. Simple click on the angle you want to set, hold the phone at the desired angle and "OK"
