  1. Download and Install the server on the computer that will host the presentation
  2. Add a rule to your firewall to allow communication over the network. The rule should be:
    • Allow
    • TCP traffic, in
    • Both directions, for
    • LaserPointerServer.exe
  4. Make sure that the phone and computer are on the same network. One way to ensure this is to connect the computer to a wifi hotspot hosted by your phone.
  5. Open the server application and press start.
  6. Open the app on your phone and make sure that the address at the top of the screen matches the address at the top of the server application.
  7. Press the reset button.
